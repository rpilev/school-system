# School System

**A front-end demo of a school management system.**

Features include:

* Managing teachers, students and lessons
* Associating teachers with students
* Associating students with subjects
* Grading students
* View statistics
  * Search for grades
  * Sort

Technologies used include:

* Vue.js
  * vue-router
  * vue-stash

* Bulme CSS
